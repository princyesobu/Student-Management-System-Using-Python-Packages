The Student Management System Using Python Packages is a simple Python project designed to manage student records efficiently. This project demonstrates the use of packages and modules in Python for organizing code into separate files, making the program easier to understand, maintain, and reuse.

The system allows users to:

Add student details
Display all student records
Search for a student using roll number
Exit the program safely
The project is divided into different modules such as:
add_student.py for adding records
display_student.py for displaying records
search_student.py for searching records
These modules are grouped inside a package folder, showing how Python packages help structure large programs in a clean and professional way.
This project is useful for beginners to learn:
Python packages
Functions and modules
Lists and dictionaries
Menu-driven programs
Basic project organization in Python

The project is developed using the Python programming language to demonstrate the concept of package creation and modular programming. It is a menu-driven application that stores and manages student information in a simple and efficient way. Each operation of the project is separated into different modules, which improves code organization and readability.
The system accepts user input for student details such as name, roll number, and course. The entered data is stored using Python data structures like lists and dictionaries. The program also provides searching functionality to find a student record quickly based on the roll number.
By using packages, the project follows a structured programming approach that reduces code complexity and supports easy maintenance. This project helps learners understand how real-time Python applications are divided into modules and packages for better software development practices.
